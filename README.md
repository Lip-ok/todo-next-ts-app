### Как запустить проект:

1. Скачать проект 
    ```sh
    git clone https://github.com/Lip-ok/todo-next-ts-app.git
    ```

2. Установите зависимости:
   ```sh
   npm install
   ```
3. Запустите приложение:
   ```sh
   npm run dev
   ```
4. Открыть в браузере:
   ```sh
   http://localhost:3000
   ```
